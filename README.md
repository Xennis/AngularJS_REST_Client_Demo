### Screenshots

![main](_screenshots/main.png)


## Getting started

### Requirements

* [node.js](http://nodejs.org/) is installed


### Installation

```sh
npm install
```

### Run

Run server :
```sh
node server.js
```

You can now open `http://localhost:3000/`
